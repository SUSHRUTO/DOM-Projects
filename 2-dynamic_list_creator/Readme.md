# 📝 Dynamic List Creator

An interactive list management app that allows users to **add, edit, and delete items dynamically**.

---

## 🚀 Features

* Add new items to the list
* Edit existing items
* Delete items instantly
* Real-time updates without page reload

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

---

## 📂 Files Included

* `index.html` – App layout
* `style.css` – Styling
* `script.js` – DOM logic for list operations

---

## ▶️ How to Run

1. Open the folder
2. Run `index.html` in your browser
3. Start adding and managing items

---

## 🎯 Learning Outcomes

* Handling user input
* Creating elements dynamically
* Updating and deleting DOM nodes

---
