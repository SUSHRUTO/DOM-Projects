# 🎨 Color Palette Generator

A fun project that generates **random colors and displays their hex codes**, useful for designers and developers.

---

## 🚀 Features

* Generate random colors
* Display hex and RGB color codes
* Dynamic UI updates

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

---

## 📂 Files Included

* `index.html` – Structure
* `style.css` – Styling for color display
* `script.js` – Logic for generating colors

---

## ▶️ How to Run

1. Open the folder
2. Run `index.html`
3. Click button to generate colors

---

## 🎯 Learning Outcomes

* Random value generation in JavaScript
* Updating styles dynamically
* Improving UI interactivity

## 📌 Future Improvements
