# 🌙 Dark Mode Toggler

A simple and interactive project that allows users to switch between **light mode and dark mode** using JavaScript DOM manipulation.

---

## 🚀 Features

* Toggle between light and dark themes
* Smooth UI experience
* Real-time DOM updates

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

---

## 📂 Files Included

* `index.html` – Structure of the webpage
* `style.css` – Styling for light/dark themes
* `script.js` – Logic for toggling themes

---

## ▶️ How to Run

1. Open the folder
2. Double-click on `index.html`
3. Click the toggle button to switch themes

---

## 🎯 Learning Outcomes

* DOM element selection
* Event handling
* Dynamic style updates
